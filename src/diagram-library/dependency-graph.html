Arrows point from the dependent module to the module it depends on. Circular dependencies are architectural smells and should be eliminated.

Notes

Dependency Graphs reveal the coupling structure of a codebase. High-fanout nodes (many dependents) are risky to change. High-fanin nodes (many dependencies) are complex to test. Use this diagram to identify candidates for extraction, inversion, or interface abstraction.
