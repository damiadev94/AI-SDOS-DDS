Component

Components are the major structural building blocks inside a container. Each component represents a grouping of related functionality with a well-defined interface.

Notes

A Component Diagram zooms into a single container and shows the named components inside it, their responsibilities, and how they interact. It is typically created by architects or senior developers to guide implementation teams. Avoid showing every class — focus on architecturally significant groupings.
