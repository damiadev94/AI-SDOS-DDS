Overview

The Export System is a planned Phase 3 capability that will provide tooling to output DDS pages and diagrams in portable formats. The goal is to allow stakeholders to receive documentation as standalone PDF files or diagram assets as SVG files, without needing access to the live HTML system. No export functionality is implemented in the current v0.1 release. Browser print-to-PDF is available as a manual workaround via the included print.css layer.

Status

Pending

The Export System has not been started. It is scoped for Phase 3, after all documentation pages and diagram content are complete and stable. The implementation approach (scripted build step vs. manual browser export) has not been decided.

Notes

Until the Export System is implemented, the recommended approach for PDF output is to use the browser's built-in print dialog (Ctrl+P / Cmd+P) on any DDS page. The print.css layer is included on every page and optimizes layout and typography for printed output.
