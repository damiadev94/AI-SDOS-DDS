Overview

This page demonstrates DDS components, typography, and layout patterns in context. Each section shows a rendered example alongside the classes and structure used to produce it. Use these examples as a reference when composing new documentation pages or adapting existing templates.

Typography

The type scale provides six levels of text hierarchy. Apply them via utility classes to maintain visual consistency across all pages.

Heading 1 — text-h2

Heading 2 — text-h3

Heading 3 — text-h4

Body text — text-body. Used for descriptive paragraphs and main content blocks.

Small text — text-small. Used for supplementary information, table cells, and code references.

Caption text — text-caption. Used for metadata labels, timestamps, and field identifiers.
